CodePills:

A super simple way of displaying web dev related tech in pill form.
I made this for a project I'm working on, and thought it might be useful to others. Please feel free to contribute or raise any issues.

Default:

With no arguments, Codepills will output all available pills with no styling attached.

Example:

createPills()

Generates:

Options and Config

Configuring pills is simple, the createPills function takes two arguments. An array and an Object. Both are optional.

createPills(ArrayOfBrands, OptionsObject)

ArrayOfBrands

The first argument for createPills is an array of strings. The strings should each be a brand name. For example "html5" or "css". 

const arrayofBrands = ['vuejs','digitalocean','vite','gatsby','tailwindcss','netlify','php']

If an empty array is passed as the first argument, createPills will return the full list of Pills. Useful for debugging.

OptionsObject

The second argument, is an Object with the following OPTIONAL attributes

Property Type Description Default Supported Values
Rounded Boolean Apply rounded corners to the pills false true,false
Spacing String Set the margin around each item 'small' 'small', 'medium', 'large'
Alignment Alignment Set the alignment of the items 'left' 'left', 'center', 'centre', 'large'

Specific list:

Passing in an array of strings to Codepills will generate pills based on that array.

Example:

const exampleBrands = ['vuejs', 'digitalocean', 'vite', 'gatsby', 'tailwindcss', 'netlify', 'php']
createPills(exampleBrands)

Rounded list:

You may enable, rounding, by passing the second argument as true. Default is false.

Example:

createPills([], {rounded: true})

Spacing Example:

Spacing changes the spacing between each item. It may be 'small', 'medium', or 'large'

Example:

createPills([],{ spacing: 'large'})

Alignment Example:

Alignment changes the alignment of the container for the pills.
It may be 'left', 'center', 'centre', or 'right'

These examples, all use a predefined array of brand names, to simplify the demo.
